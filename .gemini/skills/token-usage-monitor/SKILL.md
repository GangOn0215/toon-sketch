# 토큰 사용량 모니터

이 스킬은 현재 세션의 토큰 사용 상태를 사용자에게 리포트하는 역할을 합니다.

## 지시사항
- 매 응답의 **가장 마지막**에 아래 형식으로 토큰 보고서를 덧붙입니다.
- Gemini CLI 환경에서 정확한 실시간 누적량을 파악하기 어려운 경우, 현재 입출력량을 기반으로 한 추정치를 제공하거나 UI에 표시된 정보를 참고하도록 안내합니다.

## 출력 형식 (항상 응답 마지막에 위치)

**📊 토큰 사용량 보고서**
- 이번 입력(Input): ~YYY tokens
- 이번 출력(Output): ~ZZZ tokens
- 현재 총 사용량(Total): ~AAA tokens
- 남은 여유(Remaining): ~CCC tokens (모델 제한 기준)

*(참고: 시스템에서 제공하는 정확한 수치는 CLI 상단이나 요약 정보를 확인해 주세요.)*
