### ⚠️ [Issue] CoolSMS 에러 3059 - 발신번호 변작 차단

**증상**
```
이통사 실패 리포트: 상태코드 3059 - 번호도용/변작방지 차단
연동처 결과: 114 - 번호도용/변작방지 차단 전송 실패
```

**원인**
- CoolSMS 콘솔에 발신번호 사전등록이 되지 않은 상태
- 국내 이통사는 미등록 발신번호 SMS 전량 차단 (2015년 이후)

**해결 방법**
1. [console.solapi.com](https://console.solapi.com) 로그인
2. 좌측 메뉴 → **발신번호 관리 → 발신번호 등록**
3. 사용할 번호 입력 후 인증 (휴대폰 or ARS)
4. 상태 **"등록완료"** 확인
5. `COOLSMS_FROM_NUMBER` 환경변수와 등록 번호 일치 여부 재확인

**현재 상태**: 발신번호 등록 진행 중 (미완료)
