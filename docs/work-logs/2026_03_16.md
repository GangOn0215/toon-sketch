### ✅ [Task 9] - Supabase Auth Lock 경합 오류 해결 (13:30)
- **파일**: `src/utils/supabase/client.ts`
- **증상**: 브라우저 콘솔에서 `Lock was not released within 5000ms` 경고 발생.
- **원인**: React Strict Mode 또는 중복된 컴포넌트 마운트 시 `createBrowserClient`가 매번 새로운 인스턴스를 생성하면서 인증 토큰 접근 락(Auth Token Lock) 경합 발생.
- **해결**:

  - 서버 사이드(SSR)와 브라우저 환경을 분기 처리하여 하이드레이션 오류 방지.
  - 이제 불필요한 인스턴스 생성이 억제되어 락 경합 및 성능 저하 방지.
